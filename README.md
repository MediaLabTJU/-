## 团队简介

团队依托天津大学电气自动化与信息工程学院，主要从事多维信号表示理论与深度学习技术驱动的计算机视觉、音视频重建与分析、智能成像等领域的相关研究。近年来承担了10余项国家级与省部级项目，在领域内权威期刊与会议（例如IEEE TIP, TMM, TCSVT, CVPR, ECCV）发表论文150余篇。团队毕业生去向包括985、211高校教师，百度、华为、腾讯、阿里等知名企业算法/开发工程师，读博深造等。
